# Warehouse Management App

A simple desktop Warehouse Management System built with Python, Tkinter, and SQLite.  
This application allows users to manage products, update stock, and process sales through a graphical interface.

---

## Features

- Add new products
- Delete existing products
- Update product quantity
- Sell products
- Display all products in a table
- SQLite database integration
- User-friendly Tkinter GUI

---

## Technologies Used

- Python 3
- Tkinter
- SQLite
- ttk (Treeview)
